# Git Conventions

## Commit Messages
- Format: Conventional commits
- Types: feat, fix, chore, docs, test, refactor
- Examples:
  - `feat: Add new agent for database migrations`
  - `fix: Correct tool selection in HR manager`
  - `chore: Bump version to 0.3.0`

## Branch Naming
- feature/[description]
- fix/[description]
- hotfix/[description]
- release/[version]
- chore/[description]
